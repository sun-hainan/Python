### 2.1 线性结构

#### 数组（Array）

**原理**：连续内存空间，索引 O(1) 访问

```
内存布局：
Index:   [0]   [1]   [2]   [3]   [4]
Value:   [10]  [20]  [30]  [40]  [50]
Address: 0x100 0x104 0x108 0x10C 0x110

特点：
- 优点：索引快 O(1)、CPU缓存友好
- 缺点：插入删除慢 O(n)、大小固定
```

**操作复杂度**：

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 随机访问 | O(1) | arr[i] |
| 尾部插入 | O(1) |  amortized |
| 任意位置插入 | O(n) | 需移动元素 |
| 尾部删除 | O(1) | |
| 任意位置删除 | O(n) | 需移动元素 |

**代码实现**：

```python
class Array:
    """简单数组实现（用于理解原理）"""

    def __init__(self, capacity: int = 10):
        self.capacity = capacity
        self.size = 0
        self.data = [None] * capacity

    def get(self, index: int):
        """O(1) 获取"""
        if index < 0 or index >= self.size:
            raise IndexError("Index out of range")
        return self.data[index]

    def set(self, index: int, value) -> None:
        """O(1) 设置"""
        if index < 0 or index >= self.size:
            raise IndexError("Index out of range")
        self.data[index] = value

    def insert(self, index: int, value) -> None:
        """O(n) 插入"""
        if self.size >= self.capacity:
            self._resize(self.capacity * 2)

        # 从后往前搬移
        for i in range(self.size, index, -1):
            self.data[i] = self.data[i - 1]

        self.data[index] = value
        self.size += 1

    def delete(self, index: int):
        """O(n) 删除"""
        if index < 0 or index >= self.size:
            raise IndexError("Index out of range")

        # 从前往后搬移
        for i in range(index, self.size - 1):
            self.data[i] = self.data[i + 1]

        self.size -= 1

    def append(self, value) -> None:
        """O(1) amortized 尾部添加"""
        if self.size >= self.capacity:
            self._resize(self.capacity * 2)
        self.data[self.size] = value
        self.size += 1

    def _resize(self, new_capacity) -> None:
        """扩容"""
        new_data = [None] * new_capacity
        for i in range(self.size):
            new_data[i] = self.data[i]
        self.data = new_data
        self.capacity = new_capacity
```

#### 链表（Linked List）

**原理**：节点包含数据和指针，通过指针连接

```
单向链表：
head -> [value:10 | next] -> [value:20 | next] -> [value:30 | next] -> None

双向链表：
None <- [prev | value:10 | next] <-> [prev | value:20 | next] <-> [prev | value:30 | next] -> None
```

**数组 vs 链表**：

| 特性 | 数组 | 链表 |
|------|------|------|
| 访问 | O(1) | O(n) |
| 头部插入 | O(n) | O(1) |
| 尾部插入 | O(1) amortized | O(1) |
| 中间插入 | O(n) | O(1) 只要找到位置 |
| 内存连续 | ✅ | ❌ |
| 缓存友好 | ✅ | ❌ |

**代码实现（单向链表）**：

```python
class ListNode:
    """链表节点"""
    def __init__(self, value):
        self.value = value
        self.next = None


class LinkedList:
    """单向链表"""

    def __init__(self):
        self.head = None
        self.size = 0

    def insert_head(self, value) -> None:
        """O(1) 头部插入"""
        node = ListNode(value)
        node.next = self.head
        self.head = node
        self.size += 1

    def insert_tail(self, value) -> None:
        """O(1) 尾部插入（需要尾指针）或 O(n)"""
        node = ListNode(value)
        if not self.head:
            self.head = node
        else:
            current = self.head
            while current.next:
                current = current.next
            current.next = node
        self.size += 1

    def find(self, value):
        """O(n) 查找"""
        current = self.head
        while current:
            if current.value == value:
                return current
            current = current.next
        return None

    def delete(self, value) -> bool:
        """O(n) 删除第一个匹配项"""
        if not self.head:
            return False

        if self.head.value == value:
            self.head = self.head.next
            self.size -= 1
            return True

        current = self.head
        while current.next:
            if current.next.value == value:
                current.next = current.next.next
                self.size -= 1
                return True
            current = current.next

        return False
```

#### 栈（Stack）

**原理**：后进先出（LIFO）

```
栈操作示意图：

push(1) push(2) push(3)
    ↓      ↓      ↓
  [1]    [2]    [3]
                 ↑
               top

pop() 返回 3
```

**应用场景**：

| 场景 | 说明 |
|------|------|
| 函数调用栈 | 递归调用保存返回地址 |
| 表达式求值 | 括号匹配、中缀转后缀 |
| 深度优先搜索 | 记忆搜索路径 |
| 单调栈 | 求下一个更大/更小元素 |

**代码实现**：

```python
class Stack:
    """栈实现"""
    def __init__(self):
        self._items = []

    def push(self, item) -> None:
        """O(1) 压栈"""
        self._items.append(item)

    def pop(self):
        """O(1) 弹栈"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self._items.pop()

    def peek(self):
        """O(1) 查看栈顶"""
        if self.is_empty():
            raise IndexError("Stack is empty")
        return self._items[-1]

    def is_empty(self) -> bool:
        return len(self._items) == 0

    def size(self) -> int:
        return len(self._items)
```

**典型应用：括号匹配**

```python
def is_valid(s: str) -> bool:
    """判断括号是否匹配"""
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}

    for char in s:
        if char in mapping:
            # 右括号：检查栈顶
            if not stack or stack[-1] != mapping[char]:
                return False
            stack.pop()
        else:
            # 左括号：入栈
            stack.append(char)

    return len(stack) == 0
```

#### 队列（Queue）

**原理**：先进先出（FIFO）

```
队列操作示意图：

enqueue(1) enqueue(2) enqueue(3)
    ↓          ↓          ↓
[1, 2, 3]  [2, 3]     [3]

front →            ← rear
```

**代码实现（单向队列）**：

```python
from collections import deque

# 实际使用推荐 deque，O(1) 两端操作
queue = deque()
queue.append(1)      # O(1) 入队
queue.append(2)
item = queue.popleft()  # O(1) 出队
```

**队列变种**：

| 变种 | 特点 | 应用 |
|------|------|------|
| 双端队列 deque | 两端都可 O(1) 操作 | 滑动窗口 |
| 循环队列 | 数组实现，不需搬移 | 生产者消费者 |
| 优先队列 | 出队顺序按优先级 | Dijkstra |

### 2.2 哈希结构

#### 哈希表（Hash Table）

**原理**：通过哈希函数将键映射到数组索引

```
哈希函数: h(key) = key % 7

键:    [25, 18, 36, 9, 4, 16]
哈希值: [4,   4,  1, 2, 6, 2]

数组: [0] [1] [2] [3] [4] [5] [6]
       ↑           ↑  ↑     ↑
      36          9  16    25
                   4  18(冲突)

冲突处理方法：
1. 链地址法（拉链法）：冲突的键存在链表
2. 开放地址法：探测下一个空位
```

**操作复杂度**：

| 操作 | 平均 | 最坏 |
|------|------|------|
| 插入 | O(1) | O(n) |
| 查找 | O(1) | O(n) |
| 删除 | O(1) | O(n) |

**代码实现（链地址法）**：

```python
class HashMap:
    """哈希表（链地址法）"""

    def __init__(self, capacity: int = 16, load_factor: float = 0.75):
        self.capacity = capacity
        self.size = 0
        self.load_factor = load_factor
        self.buckets = [[] for _ in range(capacity)]

    def _hash(self, key) -> int:
        """哈希函数"""
        return hash(key) % self.capacity

    def put(self, key, value) -> None:
        """插入/更新"""
        index = self._hash(key)
        bucket = self.buckets[index]

        for i, (k, v) in enumerate(bucket):
            if k == key:
                bucket[i] = (key, value)
                return

        bucket.append((key, value))
        self.size += 1

        # 扩容检查
        if self.size > self.capacity * self.load_factor:
            self._resize()

    def get(self, key):
        """获取"""
        index = self._hash(key)
        bucket = self.buckets[index]

        for k, v in bucket:
            if k == key:
                return v

        return None

    def _resize(self) -> None:
        """扩容"""
        old_buckets = self.buckets
        self.capacity *= 2
        self.buckets = [[] for _ in range(self.capacity)]
        self.size = 0

        for bucket in old_buckets:
            for k, v in bucket:
                self.put(k, v)
```

### 2.3 树形结构

#### 二叉树（Binary Tree）

**原理**：每个节点最多两个子节点

```
二叉树示例：

         1
        / \
       2   3
      / \   \
     4   5   6

遍历顺序：
- 前序(根左右):  1, 2, 4, 5, 3, 6
- 中序(左根右):  4, 2, 5, 1, 3, 6
- 后序(左右根):  4, 5, 2, 6, 3, 1
- 层序:         1, 2, 3, 4, 5, 6
```

**代码实现**：

```python
class TreeNode:
    def __init__(self, value):
        self.value = value
        self.left = None
        self.right = None


class BinaryTree:
    """二叉树"""

    def __init__(self):
        self.root = None

    def preorder(self, node):
        """前序遍历：根左右"""
        if not node:
            return []
        return [node.value] + self.preorder(node.left) + self.preorder(node.right)

    def inorder(self, node):
        """中序遍历：左根右"""
        if not node:
            return []
        return self.inorder(node.left) + [node.value] + self.inorder(node.right)

    def postorder(self, node):
        """后序遍历：左右根"""
        if not node:
            return []
        return self.postorder(node.left) + self.postorder(node.right) + [node.value]

    def level_order(self):
        """层序遍历（广度优先）"""
        if not self.root:
            return []

        result = []
        queue = [self.root]

        while queue:
            node = queue.pop(0)
            result.append(node.value)

            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)

        return result
```

#### 二叉搜索树（BST）

**原理**：左子树 < 根 < 右子树

```
BST示例：

         8
        / \
       3   10
      / \    \
     1   6    14
        / \   /
       4   7 13

特点：
- 中序遍历是有序数组：[1, 3, 4, 6, 7, 8, 10, 13, 14]
- 查找/插入/删除：平均 O(log n)
- 最坏 O(n)（树退化成链表）
```

**关键操作代码**：

```python
class BSTNode:
    def __init__(self, value):
        self.value = value
        self.left = None
        self.right = None


class BST:
    """二叉搜索树"""

    def __init__(self):
        self.root = None

    def insert(self, value) -> None:
        """O(log n) 平均插入"""
        self.root = self._insert_recursive(self.root, value)

    def _insert_recursive(self, node, value):
        if not node:
            return BSTNode(value)

        if value < node.value:
            node.left = self._insert_recursive(node.left, value)
        else:
            node.right = self._insert_recursive(node.right, value)

        return node

    def search(self, value):
        """O(log n) 平均查找"""
        node = self.root
        while node:
            if value == node.value:
                return node
            elif value < node.value:
                node = node.left
            else:
                node = node.right
        return None
```

#### 堆（Heap）

**原理**：完全二叉树，满足堆性质

```
最大堆示例：

         9
        / \
       7   8
      / \  /
     5   6 4

性质：
- 父节点 >= 子节点（最大堆）
- 可以用数组存储
- 堆顶是最大/最小值
```

**代码实现（最大堆）**：

```python
class MaxHeap:
    """最大堆"""

    def __init__(self):
        self.heap = []

    def parent(self, i) -> int:
        return (i - 1) // 2

    def left_child(self, i) -> int:
        return 2 * i + 1

    def right_child(self, i) -> int:
        return 2 * i + 2

    def insert(self, value) -> None:
        """O(log n) 插入"""
        self.heap.append(value)
        self._sift_up(len(self.heap) - 1)

    def _sift_up(self, i) -> None:
        """向上调整"""
        while i > 0:
            p = self.parent(i)
            if self.heap[i] > self.heap[p]:
                self.heap[i], self.heap[p] = self.heap[p], self.heap[i]
                i = p
            else:
                break

    def extract_max(self):
        """O(log n) 弹出堆顶"""
        if not self.heap:
            raise IndexError("Heap is empty")

        max_val = self.heap[0]
        last = self.heap.pop()

        if self.heap:
            self.heap[0] = last
            self._sift_down(0)

        return max_val

    def _sift_down(self, i) -> None:
        """向下调整"""
        n = len(self.heap)
        while True:
            largest = i
            l = self.left_child(i)
            r = self.right_child(i)

            if l < n and self.heap[l] > self.heap[largest]:
                largest = l
            if r < n and self.heap[r] > self.heap[largest]:
                largest = r

            if largest != i:
                self.heap[i], self.heap[largest] = self.heap[largest], self.heap[i]
                i = largest
            else:
                break
```

### 2.4 图结构

#### 图的存储

| 存储方式 | 适用场景 | 时间复杂度 |
|---------|---------|-----------|
| 邻接矩阵 | 稠密图、频繁判断两点是否相邻 | O(1) 查边 |
| 邻接表 | 稀疏图、遍历邻居 | O(V+E) 遍历 |
| 链式前向星 | 中等规模 | O(V+E) |

**代码实现（邻接表）**：

```python
class Graph:
    """图（邻接表）"""

    def __init__(self, n: int, directed: bool = False):
        self.n = n
        self.directed = directed
        self.adj = [[] for _ in range(n)]

    def add_edge(self, u: int, v: int, weight: int = 1) -> None:
        """添加边"""
        self.adj[u].append((v, weight))
        if not self.directed:
            self.adj[v].append((u, weight))

    def bfs(self, start: int):
        """广度优先搜索"""
        visited = [False] * self.n
        queue = [start]
        visited[start] = True
        result = []

        while queue:
            node = queue.pop(0)
            result.append(node)

            for neighbor, _ in self.adj[node]:
                if not visited[neighbor]:
                    visited[neighbor] = True
                    queue.append(neighbor)

        return result

    def dfs(self, start: int):
        """深度优先搜索"""
        visited = [False] * self.n
        result = []

        def _dfs(node):
            visited[node] = True
            result.append(node)
            for neighbor, _ in self.adj[node]:
                if not visited[neighbor]:
                    _dfs(neighbor)

        _dfs(start)
        return result
```

## 📊 数据结构选择指南

| 需求场景 | 推荐数据结构 | 原因 |
|---------|-------------|------|
| 快速查找 O(1) | 数组/哈希表 | 随机访问 |
| 有序遍历 | BST/堆 | 自动排序 |
| 频繁头尾操作 | deque | 两端 O(1) |
| 嵌套结构递归 | 树/图 | 天然递归 |
| 先进先出 | 队列 | FIFO |
| 后进先出 | 栈 | LIFO |
| 求第 k 大/小 | 堆 | O(log n) |
