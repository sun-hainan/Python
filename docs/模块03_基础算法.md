### 3.1 排序算法

#### 八大排序算法总览

| 算法 | 平均 | 最坏 | 空间 | 稳定 |
|------|------|------|------|------|
| 冒泡排序 | O(n²) | O(n²) | O(1) | ✅ |
| 选择排序 | O(n²) | O(n²) | O(1) | ❌ |
| 插入排序 | O(n²) | O(n²) | O(1) | ✅ |
| 希尔排序 | O(n^1.3) | O(n²) | O(1) | ❌ |
| 归并排序 | O(n log n) | O(n log n) | O(n) | ✅ |
| 快速排序 | O(n log n) | O(n²) | O(log n) | ❌ |
| 堆排序 | O(n log n) | O(n log n) | O(1) | ❌ |
| 桶排序 | O(n+k) | O(n²) | O(n+k) | ✅ |

**选择建议**：
```
数据基本有序     -> 插入排序 O(n)
数据随机        -> 快速排序 O(n log n)
要求稳定排序    -> 归并排序 O(n log n)
嵌入式/内存有限  -> 堆排序 O(n log n)
特定分布数据    -> 桶排序 O(n)
```

#### 冒泡排序（最简单）

```python
def bubble_sort(arr):
    """
    冒泡排序

    原理：相邻元素比较，大值冒泡到右侧
    时间：O(n²)
    空间：O(1)
    稳定：✅
    """
    n = len(arr)

    for i in range(n):
        swapped = False  # 优化：提前终止

        for j in range(n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True

        if not swapped:  # 已经有序
            break

    return arr
```

#### 插入排序（基本有序时最快）

```python
def insertion_sort(arr):
    """
    插入排序

    原理：类似摸牌，将新牌插入已排序区域正确位置
    时间：O(n²)，数据有序时 O(n)
    空间：O(1)
    稳定：✅
    """
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1

        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1

        arr[j + 1] = key

    return arr
```

#### 归并排序（分治经典）

```python
def merge_sort(arr):
    """
    归并排序

    原理：分而治之，先分割再合并
    时间：O(n log n)
    空间：O(n)
    稳定：✅
    """
    if len(arr) <= 1:
        return arr

    # -------- 分 --------
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    # -------- 治 --------
    return merge(left, right)


def merge(left, right):
    """合并两个有序数组"""
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result
```

#### 堆排序（原地高效）

```python
def heap_sort(arr):
    """
    堆排序

    原理：构建最大堆，依次取出堆顶
    时间：O(n log n)
    空间：O(1)
    稳定：❌
    """
    n = len(arr)

    # -------- 构建最大堆 --------
    for i in range(n // 2 - 1, -1, -1):
        _sift_down(arr, i, n)

    # -------- 逐个取出堆顶 --------
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]  # 堆顶移到末尾
        _sift_down(arr, 0, i)


def _sift_down(arr, i, n):
    """向下调整"""
    while True:
        largest = i
        left = 2 * i + 1
        right = 2 * i + 2

        if left < n and arr[left] > arr[largest]:
            largest = left
        if right < n and arr[right] > arr[largest]:
            largest = right

        if largest != i:
            arr[i], arr[largest] = arr[largest], arr[i]
            i = largest
        else:
            break
```

### 3.2 二分查找

#### 标准二分

```python
def binary_search(arr, target):
    """
    二分查找（标准版）

    前提：数组有序
    时间：O(log n)
    空间：O(1)
    """
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = left + (right - left) // 2  # 防止溢出

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1  # 未找到
```

#### 二分查找变形

| 题目 | 代码模式 |
|------|---------|
| 找第一个 >= target 的位置 | `arr[mid] < target → left = mid + 1` |
| 找最后一个 <= target 的位置 | `arr[mid] > target → right = mid - 1` |
| 找第一个 > target 的位置 | `arr[mid] <= target → left = mid + 1` |
| 找最后一个小于 target 的位置 | `arr[mid] >= target → right = mid - 1` |

**例：找第一个 >= target**

```python
def lower_bound(arr, target):
    """找第一个 >= target 的索引"""
    left, right = 0, len(arr)

    while left < right:
        mid = left + (right - left) // 2

        if arr[mid] < target:
            left = mid + 1
        else:
            right = mid

    return left
```

### 3.3 双指针

#### 两数之和（有序数组）

```python
def two_sum_sorted(arr, target):
    """
    有序数组两数之和

    思路：左右指针向中间收缩
    时间：O(n)
    空间：O(1)
    """
    left, right = 0, len(arr) - 1

    while left < right:
        s = arr[left] + arr[right]

        if s == target:
            return [left, right]
        elif s < target:
            left += 1
        else:
            right -= 1

    return []
```

#### 删除重复项

```python
def remove_duplicates(nums):
    """
    删除有序数组中的重复项（返回新长度）

    思路：快慢指针
    - 快指针：遍历数组
    - 慢指针：指向有效部分的末尾
    """
    if not nums:
        return 0

    slow = 0
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow]:
            slow += 1
            nums[slow] = nums[fast]

    return slow + 1
```

### 3.4 滑动窗口

#### 最大子序和

```python
def max_subarray_kadane(nums):
    """
    最大子序和（Kadane算法）

    思路：滑动窗口的变体
    时间：O(n)
    空间：O(1)
    """
    max_sum = nums[0]
    current_sum = nums[0]

    for num in nums[1:]:
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)

    return max_sum
```

#### 长度最小的子数组

```python
def min_subarray_len(target, nums):
    """
    找长度最小的子数组，其和 >= target

    思路：滑动窗口
    时间：O(n)
    空间：O(1)
    """
    n = len(nums)
    left = 0
    current_sum = 0
    result = float('inf')

    for right in range(n):
        current_sum += nums[right]

        while current_sum >= target:
            result = min(result, right - left + 1)
            current_sum -= nums[left]
            left += 1

    return result if result != float('inf') else 0
```

### 3.5 前缀和

#### 区域和检索

```python
class PrefixSum:
    """一维前缀和"""

    def __init__(self, nums):
        self.prefix = [0]
        for num in nums:
            self.prefix.append(self.prefix[-1] + num)

    def query(self, left, right):
        """求 [left, right] 的和"""
        return self.prefix[right + 1] - self.prefix[left]
```

#### 和为 K 的子数组

```python
def subarray_sum(nums, k):
    """
    和为 k 的子数组个数

    思路：前缀和 + 哈希表
    时间：O(n)
    空间：O(n)
    """
    prefix_sum = 0
    count = 0
    prefix_count = {0: 1}  # 前缀和: 出现次数

    for num in nums:
        prefix_sum += num

        # 找 (prefix_sum - k) 出现的次数
        if prefix_sum - k in prefix_count:
            count += prefix_count[prefix_sum - k]

        # 记录当前前缀和
        prefix_count[prefix_sum] = prefix_count.get(prefix_sum, 0) + 1

    return count
```
