## 📖 单算法教程标准

每个算法知识点严格遵循 **7段式结构**：

```
1. 场景引入（为什么需要这个算法？）
2. 核心原理通俗讲解（图文并茂，大白话）
3. 底层深度剖析（为什么这样设计？）
4. 完整规范代码（可直接运行）
5. 关键细节&易错点（高频错误汇总）
6. 经典例题分层训练（入门→中档→拔高）
7. 总结复盘&拓展延伸（对比、变种、延伸）
```
