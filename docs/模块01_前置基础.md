### 1.2 代码前置统一规范

**Python 编码规范**：

```python
# ✅ 正确示例
def binary_search(arr: list, target: int) -> int:
    """
    二分查找

    参数:
        arr: 有序数组
        target: 目标值

    返回:
        目标索引，不存在返回-1
    """
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = left + (right - left) // 2  # 防止溢出

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1
```

### 1.4 复杂度分析（重中之重）

#### 时间复杂度

**定义**：随着输入规模 n 增长，算法执行时间的变化趋势

**常见复杂度等级**：

| 复杂度 | 名称 | 10个元素 | 100个元素 | 1000个元素 |
|--------|------|----------|-----------|------------|
| O(1) | 常数 | 1 | 1 | 1 |
| O(log n) | 对数 | 3 | 7 | 10 |
| O(n) | 线性 | 10 | 100 | 1000 |
| O(n log n) | 线性对数 | 33 | 664 | 9966 |
| O(n²) | 平方 | 100 | 10000 | 1000000 |
| O(2^n) | 指数 | 1024 | 2^100 | 2^1000 |

**复杂度对比图**：

```
增长趋势（从快到慢）

O(2^n)     ████████████████████████████ 极快增长
O(n!)      ████████████████████████
O(n³)      ████████████
O(n²)      ████████
O(n log n) █████
O(n)       ████
O(log n)   ██
O(1)       █
```

#### 空间复杂度

**定义**：算法所需内存空间与输入规模的关系

| 空间复杂度 | 说明 |
|-----------|------|
| O(1) | 原地算法，只用常数个额外变量 |
| O(n) | 线性额外空间 |
| O(n²) | 二维数组等 |

#### 计算技巧

**方法1：数循环次数**

```python
# 单层循环 -> O(n)
for i in range(n):
    ...

# 双层循环（无关） -> O(n²)
for i in range(n):
    for j in range(n):
        ...

# 双层循环（有关） -> O(n)
for i in range(n):
    for j in range(i):  # j < i
        ...
```

**方法2：主定理（递归复杂度）**

```
T(n) = aT(n/b) + O(n^d)

如果 n^(log_b a) > n^d，则 T(n) = O(n^(log_b a))
如果 n^(log_b a) = n^d，则 T(n) = O(n^d * log n)
如果 n^(log_b a) < n^d，则 T(n) = O(n^d)
```

**常见递归复杂度**：

| 递归式 | 复杂度 | 经典算法 |
|--------|--------|---------|
| T(n)=2T(n/2)+O(1) | O(n) | 二分归并 |
| T(n)=2T(n/2)+O(n) | O(n log n) | 归并排序 |
| T(n)=T(n/2)+O(1) | O(log n) | 二分查找 |
| T(n)=2T(n-1)+O(1) | O(2^n) | 递归斐波那契 |

#### 复杂度分析实战

**例题1：冒泡排序**

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):          # 外层: n次
        for j in range(n - i - 1):  # 内层: n-1, n-2, ..., 1
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]

# 复杂度分析：
# 外层循环: n次
# 内层循环: n-1 + n-2 + ... + 1 = n(n-1)/2
# 总操作: O(n²)
```

**例题2：二分查找**

```python
def binary_search(arr, target):
    left, right = 0, len(arr) - 1

    while left <= right:  # 循环 log₂(n) 次
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

# 复杂度分析：
# 每次循环区间减半
# 循环次数: O(log n)
```
