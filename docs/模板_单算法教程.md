## 📖 章节一：场景引入（为什么需要这个算法？）

### 1.1 生活化例子

> 用生活中的例子说明：这个算法解决什么问题？

**例子（快速排序）**：
>
> 想象你有一堆扑克牌要按顺序排列。
> 你把牌分成两堆：小牌一堆，大牌一堆。
> 然后对每堆继续这个操作...
> 这就是快速排序的核心思想！

### 1.2 为什么需要这个算法？

| 问题 | 如果不用这个算法 | 用这个算法的优势 |
|------|-----------------|-----------------|
| （举例）排序太慢 | 用冒泡排序 O(n²) | 快速排序 O(n log n) |
| （举例）找不到元素 | 一个个找 O(n) | 二分查找 O(log n) |

### 1.3 什么时候用？

✅ **适用场景**：
- 数据量大、需要高效排序
- 需要找第 k 大/小的元素
- 数据需要分治处理

❌ **不适用场景**：
- 数据量很小（不如用插入排序）
- 需要稳定排序（相同值保持相对位置）

## 📖 章节二：核心原理通俗讲解

### 2.1 核心思想（一句话概括）

> 快速排序的核心：**选基准、分区、递归**

### 2.2 图文详解

```
步骤1：选择基准（pivot）
原数组: [8, 3, 1, 7, 0, 10, 2]
选基准: pivot = 5（任意选择，这里选中间）

步骤2：分区（partition）
将数组分为两部分：小于 pivot 和 大于 pivot
[3, 1, 0, 2]  pivot  [8, 7, 10]
      4              9

步骤3：对两个分区递归执行
[3, 1, 0, 2] -> [2, 0, 1] pivot=2 -> [1, 0] pivot=1 -> [0]

合并结果: [0, 1, 2, 3, 5, 7, 8, 9, 10]
```

### 2.3 关键代码解释

```python
def quick_sort(arr, left, right):
    # -------- 递归终止条件 --------
    # 当 left >= right，说明子数组只有0或1个元素
    if left >= right:
        return

    # -------- 第1步：选择基准 --------
    # 这里选择中间位置的元素
    pivot_index = (left + right) // 2
    pivot = arr[pivot_index]

    # -------- 第2步：分区 --------
    # 把小于 pivot 的放左边，大于的放右边
    i = left      # 左指针
    j = right     # 右指针

    while i <= j:
        # 左指针找到第一个 >= pivot 的元素
        while arr[i] < pivot:
            i += 1

        # 右指针找到第一个 <= pivot 的元素
        while arr[j] > pivot:
            j -= 1

        # 交换
        if i <= j:
            arr[i], arr[j] = arr[j], arr[i]
            i += 1
            j -= 1

    # -------- 第3步：递归处理左右分区 --------
    quick_sort(arr, left, j)   # 左边
    quick_sort(arr, i, right)  # 右边
```

| 对比项 | 快速排序 | 归并排序 | 堆排序 |

## 📖 章节三：底层深度剖析

### 3.1 为什么快速排序实际很快？

**原因1：缓存友好**
```
快速排序是原地排序，不需要额外数组
而且分区操作有良好的空间局部性，CPU缓存命中率高
```

**原因2：常数因子小**
```
快速排序内部操作简单：比较、交换
没有归并排序的合并操作
```

### 3.2 为什么最坏情况 O(n²)？

**场景**：数组已经有序

```
原数组: [1, 2, 3, 4, 5]
如果每次选第一个/最后一个作为基准：
- 第一次分区：基准=1，剩下 [2,3,4,5]
- 第二次分区：基准=2，剩下 [3,4,5]
...
时间复杂度 = n + (n-1) + (n-2) + ... + 1 = O(n²)
```

### 3.3 如何避免最坏情况？

**方法1：随机选择基准**

```python
import random

def quick_sort_random(arr, left, right):
    if left >= right:
        return

    # 随机选择基准，避免最坏情况
    rand_index = random.randint(left, right)
    arr[left], arr[rand_index] = arr[rand_index], arr[left]

    # 然后用原来的分区逻辑
    pivot = arr[left]
    # ... partition logic ...
```

**方法2：三数取中**

```python
def median_of_three(arr, left, right):
    """取三个点（首、中、尾）的中位数作为基准"""
    mid = (left + right) // 2

    # 确保 left <= mid <= right
    if arr[left] > arr[mid]:
        arr[left], arr[mid] = arr[mid], arr[left]
    if arr[left] > arr[right]:
        arr[left], arr[right] = arr[right], arr[left]
    if arr[mid] > arr[right]:
        arr[mid], arr[right] = arr[right], arr[mid]

    # 交换后，mid 位于 right-1，作为基准
    arr[mid], arr[right-1] = arr[right-1], arr[mid]
    return arr[right-1]
```

### 3.4 荷兰国旗问题（分区变种）

**问题**：将数组分为三部分：小于基准、等于基准、大于基准

```python
def dutch_national_flag(arr):
    """
    荷兰国旗问题
    时间复杂度: O(n)
    空间复杂度: O(1)
    """
    low = 0      # 小于区域右边界
    mid = 0      # 当前扫描位置
    high = len(arr) - 1  # 大于区域左边界
    pivot = 1     # 基准值

    while mid <= high:
        if arr[mid] < pivot:
            # 当前元素小于基准，交换到前面
            arr[low], arr[mid] = arr[mid], arr[low]
            low += 1
            mid += 1
        elif arr[mid] == pivot:
            # 当前元素等于基准，继续扫描
            mid += 1
        else:
            # 当前元素大于基准，交换到后面
            arr[mid], arr[high] = arr[high], arr[mid]
            high -= 1
            # 注意：交换后 mid 不变，因为新换来的元素还没检查
```

## 📖 章节四：完整规范代码

### 4.1 标准版（面试常用）

```python
def quick_sort(arr: list) -> list:
    """
    快速排序

    【时间复杂度】
    - 平均: O(n log n)
    - 最坏: O(n²)（数据有序时）
    - 最好: O(n log n)

    【空间复杂度】
    - O(log n)（递归栈）

    【稳定性】
    - 不稳定

    【参数】
    - arr: 待排序数组

    【返回】
    - 排序后的数组（原地修改）
    """
    def partition(arr, left, right):
        """
        分区函数

        【返回】
        - pivot 的最终位置
        """
        # -------- 选择基准（随机化避免最坏情况）--------
        import random
        rand_idx = random.randint(left, right)
        arr[left], arr[rand_idx] = arr[rand_idx], arr[left]

        pivot = arr[left]  # 选择第一个作为基准
        i, j = left + 1, right

        while True:
            # -------- 找到左边第一个 > pivot 的元素 --------
            while i <= j and arr[i] <= pivot:
                i += 1

            # -------- 找到右边第一个 < pivot 的元素 --------
            while i <= j and arr[j] >= pivot:
                j -= 1

            # -------- 退出条件 --------
            if i >= j:
                break

            # -------- 交换 --------
            arr[i], arr[j] = arr[j], arr[i]
            i += 1
            j -= 1

        # -------- 将基准放到正确位置 --------
        arr[left], arr[j] = arr[j], arr[left]

        return j

    def sort(arr, left, right):
        """递归排序"""
        # -------- 递归终止条件 --------
        if left >= right:
            return

        # -------- 分区 --------
        pivot_index = partition(arr, left, right)

        # -------- 递归处理左右两部分 --------
        sort(arr, left, pivot_index - 1)
        sort(arr, pivot_index + 1, right)

    # -------- 空数组或单元素 --------
    if not arr or len(arr) <= 1:
        return arr

    sort(arr, 0, len(arr) - 1)
    return arr
```

### 4.2 简洁版（笔试快速实现）

```python
def quick_sort(arr):
    """简洁版快速排序"""
    if len(arr) <= 1:
        return arr

    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    mid = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]

    return quick_sort(left) + mid + quick_sort(right)
```

### 4.3 原地版本（最优空间）

```python
def quick_sort_inplace(arr: list) -> None:
    """原地快速排序（不创建新数组）"""
    def sort(left, right):
        if left >= right:
            return

        pivot = arr[(left + right) // 2]
        i, j = left, right

        while i <= j:
            while arr[i] < pivot:
                i += 1
            while arr[j] > pivot:
                j -= 1
            if i <= j:
                arr[i], arr[j] = arr[j], arr[i]
                i += 1
                j -= 1

        sort(left, j)
        sort(i, right)

    if arr:
        sort(0, len(arr) - 1)
```

## 📖 章节五：关键细节&易错点

### 5.1 高频错误汇总

| 错误代码 | 错误原因 | 正确写法 |
|----------|---------|---------|
| `while i < right` | 遗漏等于 | `while i <= j` |
| `return i` | 基准位置错误 | `return j`（基准在 j 位置） |
| 忘记 `import random` | 随机化失败 | 在函数内或全局导入 |
| 递归忘记终止条件 | 无限递归 | 必须有 `if left >= right` |

### 5.2 边界情况处理

```python
# 边界情况1：空数组
quick_sort([])  # 应返回 []

# 边界情况2：单元素
quick_sort([1])  # 应返回 [1]

# 边界情况3：全部相同
quick_sort([1,1,1,1])  # 应返回 [1,1,1,1]

# 边界情况4：已经有序
quick_sort([1,2,3,4,5])  # 随机化基准后正常
```

## 📖 章节六：经典例题分层训练

### 6.1 入门题：基本排序

**题目**：排序并返回第 k 大的元素

```python
def find_kth_largest(arr, k):
    """
    利用快速排序的思想找第 k 大元素
    平均时间复杂度: O(n)

    【思路】
    分区后，基准位置 j：
    - 如果 j == k-1，找到答案
    - 如果 j > k-1，在左边继续找
    - 如果 j < k-1，在右边继续找
    """
    def partition(left, right):
        pivot = arr[left]
        i, j = left + 1, right

        while True:
            while i <= j and arr[i] >= pivot:
                i += 1
            while i <= j and arr[j] <= pivot:
                j -= 1
            if i >= j:
                break
            arr[i], arr[j] = arr[j], arr[i]
            i += 1
            j -= 1

        arr[left], arr[j] = arr[j], arr[left]
        return j

    left, right = 0, len(arr) - 1
    k = k - 1  # 转为索引

    while left <= right:
        pivot_index = partition(left, right)

        if pivot_index == k:
            return arr[pivot_index]
        elif pivot_index > k:
            right = pivot_index - 1
        else:
            left = pivot_index + 1

    return None  # 未找到
```

### 6.3 拔高题：数组中第 k 个最小的数

```python
def find_kth_smallest(arr, k):
    """
    找第 k 小的元素（1 <= k <= n）
    平均时间: O(n)
    最坏时间: O(n²)

    【思路】
    同找第 k 大，只是比较方向相反
    """
    def partition(left, right):
        pivot = arr[left]
        i, j = left + 1, right

        while i <= j:
            while i <= j and arr[i] <= pivot:
                i += 1
            while i <= j and arr[j] >= pivot:
                j -= 1
            if i < j:
                arr[i], arr[j] = arr[j], arr[i]
                i += 1
                j -= 1

        arr[left], arr[j] = arr[j], arr[left]
        return j

    left, right = 0, len(arr) - 1
    k = k - 1

    while left <= right:
        pivot_index = partition(left, right)

        if pivot_index == k:
            return arr[pivot_index]
        elif pivot_index > k:
            right = pivot_index - 1
        else:
            left = pivot_index + 1

    return None
```

## 📖 章节七：总结复盘&拓展延伸

### 7.1 一句话总结

> **快速排序 = 选基准 + 分区 + 递归**
> 核心优势：原地排序、缓存友好、常数因子小
> 关键优化：随机基准、三数取中

### 7.2 记忆口诀

```
快速排序三步走：
一选基准（随机化）
二分区（左右指针）
三递归（分而治之）
```

### 7.3 算法对比总结

| 排序算法 | 时间(平均) | 时间(最坏) | 空间 | 稳定 |
|----------|-----------|-----------|------|------|
| 快速排序 | O(n log n) | O(n²) | O(log n) | ❌ |
| 归并排序 | O(n log n) | O(n log n) | O(n) | ✅ |
| 堆排序 | O(n log n) | O(n log n) | O(1) | ❌ |
| 冒泡排序 | O(n²) | O(n²) | O(1) | ✅ |
